# Requisito para a execução:
    É necessário ter o Python instalado em seu computador!

#  Como executar?
1 - Abra o terminal no diretório onde o programa "golden-model.py" está;
2 - Execute o script com o comando python (ou python3 se estiver em um sistema onde python pode referir-se à versão 2.x). No caso, o comando inteiro será "python golden-model.py";
3 - Um arquivo "estimulos.dat" será criado se tudo ocorrer corretamente, e com isso a finalização do processo;
4 - Colocar o arquivo .dat na pasta do modelsim.
